Networking Resources

What you can do on this page

Use these resources to (1) build a networking plan, (2) send outreach messages, and (3) conduct informational interviews with confidence.

Quick Start (20 minutes)

  1. Draft a 2-3 sentence intro using the elevator pitch rubric.
  2. Pick 3 people to reach out to (alumni, speakers, classmates, or colleagues).
  3. Send one outreach message using the examples/templates below.

Build a Strategic Networking Plan

  • Intro to Making Connections (Workshop Slides) — Understand where to start and how to build connections over time.
  • Building a List for Strategic Networking (Handout) — A simple template for tracking people and follow-ups.
  • Elevator Pitch Rubric — Create a short introduction you can use in events and messages.
